# OPC 主页优化设计文档

## 目标

优化 OPC + Agent AI 原生创业工场主页，提升移动端体验，突出项目制训练营介绍，优化合作对接入口，并调整信息层级。

## 设计原则

- **保持原有排版格式**：不新增 CSS 组件，不修改视觉风格（颜色、字体、圆角、阴影等）
- **最小改动**：仅调整 HTML section 顺序和文案内容
- **移动端友好**：修复现有响应式中的细节问题
