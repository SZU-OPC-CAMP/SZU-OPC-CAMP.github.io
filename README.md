# OPC + Agent AI 原生创业工场

OPC 训练营合作展示网页，面向高校、企业和投资人展示项目机制、资源网络与学生项目库。

## 页面

- `index.html`: 合作展示首页
- `projects.html`: 项目库
- `project.html?id=<project-id>`: 项目详情页

## 本地预览

```bash
python -m http.server 5178
```

然后打开 `http://127.0.0.1:5178/index.html`。

## GitHub Pages

这是纯静态站点，可以直接通过 GitHub Pages 从 `main` 分支根目录发布。
